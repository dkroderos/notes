### Start minikube

```
minikube start --extra-config=apiserver.authorization-mode=RBAC
```

### Get services

```
kubectl get services
```

### Get pods

```
kubectl get pod
```

### Create deployment

```
kubectl create deployment nginx-depl --image=nginx
```

### Get deployments

```
kubectl get deployment
```
